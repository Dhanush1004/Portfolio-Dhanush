# Portfolio – Dhanush

A modern, animated personal portfolio showcasing my skills, projects, and experience as a Frontend and Full-Stack Developer.

This portfolio is built with a strong focus on performance, smooth animations, clean UI design, and real-world usability.

---

## About Me

I am Dhanush, a final-year engineering student and a passionate developer interested in building interactive and scalable web applications.

I enjoy working with modern frontend frameworks, animations, and backend technologies to create meaningful digital experiences.

---

## Tech Stack

### Frontend
- React
- Next.js
- TypeScript
- Tailwind CSS
- GSAP
- Framer Motion

### Backend
- Node.js
- REST APIs
- Authentication and Authorization

### Databases
- MongoDB
- MySQL

### Tools and Platforms
- Git
- GitHub
- VS Code
- Postman
- npm
- Vercel

---

## Features

- Advanced animations using GSAP and Canvas
- Interactive pixel-based hover effects
- Performance-optimized animations
- Fully responsive layout
- Clean and modern UI design
- Optional sound-based interactions
- Scroll-based animations and transitions
